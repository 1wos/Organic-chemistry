# 📄 Page 1: 친핵성 아실 치환반응 (NAS)

## 🔷 왼쪽 열: Basis of NAS

---

### 칸 1: NAS 기본 개념 (맨 위)

**비유**: 회사에서 직원 교체하는 것과 같습니다!

```
기존 직원(이탈기 X) → 해고
새 직원(친핵체 Nu) → 채용
회사(아실기) → 그대로 유지
```

**핵심 반응**:
- **Acyl group (아실기)**: C=O가 붙은 탄소 골격 (회사 본체)
- **Nucleophile (친핵체)**: 전자가 풍부해서 공격하는 분자 (새 직원)
- **Leaving group (이탈기)**: 떠나는 그룹 (해고되는 직원)

**Tetrahedral intermediate (사면체 중간체)**:
- 새 직원이 들어오는 순간, 잠깐 사람이 많아지는 상태
- 평면(sp²) → 사면체(sp³) → 다시 평면(sp²)

---

### 칸 2: 카르복실산과 알코올의 산-염기 반응

**비유**: 힘겨루기!

```
카르복실산 (pKa = 4) ←→ 알코올 (pKa = 16)
   강한 산                약한 산
```

- 카르복실산이 알코올에게 H⁺를 주면?
- 카르복실레이트(강한 염기) vs 메탄올(약한 염기) 생성
- **평형은 약한 쪽으로 이동** → 오른쪽으로 거의 안 감

---

### 칸 3: Mechanism #2 - 4단계 메커니즘

**비유**: 문을 열고(addition) → 자리 바꾸고(proton transfer) → 문 닫고(elimination) → 정리(deprotonation)

1. **[1,2] addition**: 물(H₂O)이 카르보닐에 붙음
2. **Proton transfer**: 양성자가 이동
3. **[1,2] elimination**: Cl⁻가 떠나면서 C=O 재형성
4. **Deprotonation**: 남은 H⁺ 제거

---

### 칸 4: Lactone(락톤) 반응

**락톤이란?**: 고리형 에스터 (분자 내에서 -OH와 -COOH가 연결된 것)

**NaOH와 반응하면**:
1. 친핵성 첨가 (고리가 열림)
2. 양성자 이동
3. 강한 염기가 약한 염기를 밀어냄

**결과**: 열린 사슬 형태의 하이드록시 카르복실레이트

---

## 🔷 중앙 열: Anionic & Neutral Nucleophile

---

### 칸 5: 음이온성 vs 중성 친핵체 비교

| 구분 | 음이온성 친핵체 | 중성 친핵체 |
|-----|---------------|------------|
| **예시** | ⁻OH, ⁻OR, ⁻NH₂, R-MgX, LiH | H₂O, ROH, NH₃, RNH₂ |
| **촉매** | 산 촉매 ❌ (서로 반응해버림) | 산 촉매 ⭕ (반응 가속화) |
| **가역성** | 비가역적 가능 | 대부분 평형 반응 |
| **마지막 단계** | "산 추가" (acid workup) | "탈양성자화" |

**비유**: 
- 음이온성 = 적극적인 공격수 (혼자서도 잘함)
- 중성 = 도움이 필요한 공격수 (산이 도와줘야 함)

---

### 칸 6: 중성 친핵체의 반응 메커니즘

**산 촉매가 하는 일**:
1. 카르보닐 산소를 양성자화 → 탄소가 더 (+)성을 띔
2. 약한 친핵체도 공격 가능해짐

**Hemiacetal (헤미아세탈)**: 알코올이 알데히드에 1번 붙은 중간체
**Enamine (엔아민)**: 아민이 붙고 물이 빠진 형태

---

### 칸 7: pKa 다이어그램 - 친핵체 vs 이탈기

**핵심 원리**: 
> "pKa가 높으면 강한 염기 = 좋은 친핵체, 나쁜 이탈기"
> "pKa가 낮으면 약한 염기 = 나쁜 친핵체, 좋은 이탈기"

```
pKa 50 ← 매우 강한 염기 (CH₃⁻, NH₂⁻)
pKa 30 ← 강한 염기
pKa 10 ← 중간 (RO⁻, HO⁻)
pKa 0  ← 약한 염기 (Cl⁻) = 좋은 이탈기!
pKa -10 ← 매우 약한 염기
```

**"Downhill" 규칙**: 반응은 항상 pKa가 높은 쪽 → 낮은 쪽으로 진행

---

### 칸 8: LiAlH₄ (리튬 알루미늄 하이드라이드)

**비유**: 최강의 환원제! 모든 것을 알코올로 만드는 "만능 환원기"

**특징**:
- H⁻ (하이드라이드)를 전달
- 매우 강력해서 물과도 격렬히 반응

**반응 결과**:
- 에스터 → **1차 알코올**
- 산염화물 → **1차 알코올**
- 아마이드 → **아민**

---

### 칸 9: DIBAL-H (다이발-H)

**비유**: "절제된" 환원제 - 딱 필요한 만큼만!

**특징**:
- -78°C (드라이아이스 온도)에서 사용
- 부피가 커서 한 번만 반응하고 멈춤

**반응 결과**:
- 에스터 → **알데히드** (완전히 환원 안 됨!)

**LiAlH₄와 차이점**: LiAlH₄는 알코올까지, DIBAL-H는 알데히드에서 멈춤

---

### 칸 10: Grignard 시약 (그리냐르)

**비유**: R 그룹을 2개 던져주는 "더블 펀치"

**구조**: R-MgBr (탄소에 (-)전하, 마그네슘에 (+)전하)

**메커니즘 (Double Addition)**:
1. 첫 번째 R 공격 → 케톤 중간체
2. 두 번째 R 공격 → 알코올
3. 산 처리 (acid workup)

**결과**: **3차 알코올** (탄소에 3개의 R 그룹)

---

### 칸 11: Gilman 시약

**비유**: "한 방만" 때리는 정밀 타격

**구조**: R₂CuLi (리튬 다이알킬큐프레이트)

**특징**:
- 산염화물과만 반응
- R 그룹 1개만 전달

**결과**: **케톤** (알코올까지 안 감!)

---

## 🔷 오른쪽 열: 5 Mechanism, 6 Product, and Fischer

---

### 칸 12: P - Protonation (양성자화)

**비유**: 카르보닐에 "파워업" 시키기

```
C=O + H⁺ → C=O⁺-H
```

**특징**:
- ⁻O⁻ 는 강한 염기, ⁺OH₂는 강한 산
- 혼성화 변화 없음 (sp² 유지)
- **매우 빠른 반응** (거의 순간적)

---

### 칸 13: 12A - [1,2] Addition (1,2-첨가)

**비유**: 이중결합에 "끼어들기"

```
C=O → C-O (단일결합으로)
     |
    Nu (친핵체가 붙음)
```

**특징**:
- 친핵체가 카르보닐 탄소 공격
- sp² → sp³ 변화
- 사면체 중간체 형성

---

### 칸 14: PT - Proton Transfer (양성자 이동)

**비유**: "분자 내 양성자 택배"

**특징**:
- 같은 분자 안에서 H⁺가 이동
- 분자간 염기-산 반응으로도 가능
- C=O 이중결합 재형성 과정의 일부

---

### 칸 15: 12E - [1,2] Elimination (1,2-제거)

**비유**: 이탈기를 "밀어내고" 이중결합 복구

```
C-O⁻ → C=O + 이탈기
 |
 X (이탈기)
```

**특징**:
- C=O 이중결합 재형성
- 이탈기 탈락
- **느린 반응** (율속 단계일 수 있음)
- sp³ → sp² 변화 (기하구조 재배열 필요)

---

### 칸 16: D - Deprotonation (탈양성자화)

**비유**: 여분의 H⁺ "정리하기"

```
O-H → O⁻ + H⁺
```

**특징**:
- 기질에서 양성자 제거
- 촉매 재생성에 중요

---

### 칸 17: 6가지 카르보닐 유도체와 상호전환

**서열 (반응성 순서)**:
```
Acid chloride > Anhydride > Ester ≈ Carboxylic acid > Amide
(가장 반응성 높음)                                    (가장 안정)
```

**주요 전환**:
- 카르복실산 + ROH → **에스터** (Fischer)
- 에스터 + H₂O → **카르복실산** (가수분해)
- 아마이드 + H₂O → **카르복실산** (가수분해)
- 케톤/알데히드 + 아민 → **이민**
- 카르복실산 + 산염화물 시약 → **산무수물**

---

### 칸 18: Fischer Esterification (피셔 에스터화)

**비유**: 카르복실산 + 알코올 = 에스터 + 물 (탈수 축합)

**전체 반응**:
```
R-COOH + R'-OH ⇌ R-COOR' + H₂O
```

**5단계 메커니즘**:
1. **Protonation**: 산 촉매가 C=O 활성화
2. **Nucleophilic attack**: 알코올이 공격
3. **Proton transfer**: 양성자 재배치
4. **Elimination**: 물 탈락
5. **Deprotonation**: 촉매 재생성

**특징**: 
- 가역 반응 (평형!)
- 르샤틀리에 원리로 평형 이동 가능 (물 제거 or 알코올 과량)

# 📄 Page 2: 에놀레이트 화학

## 🔷 왼쪽 열: Enolates

---

### 칸 19: Enolate 기본 개념

**비유**: 카르보닐 옆 탄소(α-탄소)가 "변신"하는 것

```
    O           O⁻
    ‖           |
R-C-CH₃  →  R-C=CH₂
  케톤        에놀레이트
```

**α-탄소란?**: 카르보닐(C=O) 바로 옆 탄소

**왜 α-수소가 산성인가?** (4가지 이유):
1. **치환기 효과**: 전자 끄는 그룹이 있으면 더 산성
2. **공명 안정화**: 음전하가 산소로 분산
3. **수소결합**: 안정화에 기여
4. **방향족성**: 특수한 경우 추가 안정화

---

### 칸 20: Keto-Enol Tautomerism 메커니즘

**호변이성이란?**: 두 구조가 양성자 이동으로 상호전환

**산 촉매 메커니즘**:
1. 카르보닐 산소 양성자화
2. α-수소 제거 → 에놀 형성

**염기 촉매 메커니즘**:
1. α-수소 제거 → 에놀레이트
2. 산소 양성자화 → 에놀

---

### 칸 21: 에놀레이트 이온의 공명 구조

**공명으로 안정화**:
```
    ⊖O           O
     |           ‖
  R-C=CH₂ ↔ R-C-CH₂⊖
  (산소에 전하)  (탄소에 전하)
```

**결과**:
- 음전하가 O와 C에 분산
- 전자끄는 그룹(EWG)이 있으면 더 안정
- 더 전기음성인 O에 음전하가 더 많이 위치

---

### 칸 22: Kinetic vs Thermodynamic Enolate

**비유**: "빨리 가는 길" vs "안정적인 길"

| 구분 | Kinetic (속도론적) | Thermodynamic (열역학적) |
|-----|-------------------|------------------------|
| **조건** | LDA, -78°C, 짧은 시간 | NaOH/ROH, 상온, 긴 시간 |
| **선택** | 덜 치환된 쪽 | 더 치환된 쪽 |
| **이유** | 입체장애 적어서 빠름 | 이중결합이 더 안정 |
| **에너지** | 활성화 에너지 낮음 | 생성물 에너지 낮음 |

**에너지 다이어그램**:
- Kinetic: 낮은 활성화 에너지 장벽
- Thermodynamic: 더 안정한 최종 상태

## 🔷 중앙 열: Enolate Synthesis

---

### 칸 23: 에놀레이트 형성과 알킬화

**Step 1: Deprotonation (탈양성자화)**
```
        O                    O⊖
        ‖      + NaH         |
H₃C-C-CH₃  ────→    H₃C-C=CH₂  + H₂
```

**Step 2: SN2 Alkylation (알킬화)**
```
에놀레이트 + R-Br → 새로운 C-C 결합 + Br⁻
```

**핵심**: 에놀레이트의 α-탄소가 친핵체로 작용!

---

### 칸 24: α-Halo Ketone 형성

**반응**: 에놀레이트 + 할로겐(Cl₂, Br₂, I₂)

```
에놀레이트 + Br-Br → α-bromo ketone + Br⁻
```

**주의 (NaOH/H₂O 조건)**:
- 생성물이 출발물보다 더 산성!
- → 다중 할로겐화 발생 가능
- → Haloform 반응으로 이어질 수 있음

---

### 칸 25: Aldol Reaction (알돌 반응)

**비유**: 두 카르보닐이 "손잡기"

**반응**:
```
에놀레이트 + 알데히드/케톤 → β-hydroxy carbonyl
```

**메커니즘**:
1. 에놀레이트 형성 (탈양성자화)
2. 다른 카르보닐에 친핵성 첨가 (새 C-C 결합!)
3. 양성자화 → β-하이드록시 케톤/알데히드

**Aldol Condensation**: 알돌 생성물에서 물 탈락 → α,β-불포화 카르보닐

---

### 칸 26: Crossed Aldol (교차 알돌)

**문제점**: 4가지 생성물이 가능!

```
Aldehyde A + Aldehyde B
→ Self A + Self B + Crossed #1 + Crossed #2
```

**해결책**:
- 한 쪽은 에놀화 불가능한 것 사용 (α-H 없는 알데히드)
- 예: 벤즈알데히드, 포름알데히드
- 또는 한 쪽을 과량 사용

---

## 🔷 오른쪽 열: Enolate vs. Enol

---

### 칸 27: Keto-Enol Tautomerism 기본

**두 형태의 비교**:

| Keto (케토) | Enol (에놀) |
|------------|-------------|
| C=O 이중결합 | C=C 이중결합 + O-H |
| **친전자성** | **친핵성** (α-탄소에서) |
| 보통 더 안정 | 보통 덜 안정 |

**중요**: 공명이 아니라 **구조 이성질체**! (실제로 원자 위치가 다름)

---

### 칸 28: 산 촉매 할로겐화

**반응**: 케톤 + Br₂ (산 촉매) → α-bromo ketone

**메커니즘**:
1. 케톤 → 에놀 (호변이성)
2. 에놀이 Br₂ 공격
3. α-브로모 케톤 생성

---

### 칸 29: 산 촉매 알돌 반응 Part 1

**산이 하는 일**: 케토-에놀 호변이성 촉진

```
케톤 + H⁺ ⇌ 양성자화된 케톤 ⇌ 에놀 + H⁺
```

**에놀 형성**: α-탄소에서 친핵성 발현

---

### 칸 30: 산 촉매 알돌 반응 Part 2 & 3

**Part 2**: 양성자화된 카르보닐은 더 좋은 친전자체
```
C=O + H⁺ → C=O⁺H (훨씬 더 (+)성)
```

**Part 3**: 산 촉매 알돌 축합
- 물 탈락이 쉬움 (OH가 OH₂⁺로 양성자화 → 좋은 이탈기)
- α,β-불포화 카르보닐 직접 생성

---

### 칸 31: Mannich Reaction (만니히 반응)

**비유**: 알돌인데, 카르보닐 대신 **이민**에 첨가

**반응**:
```
에놀 + 이미늄 이온 → β-amino ketone
```

**이미늄 이온**: C=N⁺ (질소 버전의 활성화된 카르보닐)

**생성물**: β-아미노 케톤 (질소가 β-위치에)

---

### 칸 32: 염기 촉매 알돌 반응 전체

**완전한 메커니즘**:
1. **Step 1**: 에놀레이트 형성
2. **Step 2**: 카르보닐에 첨가 (C-C 결합 형성)
3. **Step 3**: 양성자화

**알돌 축합까지**:
4. **Step 1**: 새 에놀레이트 형성
5. **Step 2**: 탈락 반응 (E1cb) → α,β-불포화 생성물

## 🔷 왼쪽 열: Keto & Enol Named Reactions

---

### 칸 33: Mannich Reaction 상세

**정의**: "알돌과 비슷하지만 C=N에 첨가"

**필요한 것들**:
- 에놀화 가능한 케톤
- 포름알데히드 (CH₂O)
- 2차 아민

**메커니즘**:
1. 아민 + 알데히드 → 이미늄 이온
2. 에놀이 이미늄 공격
3. β-아미노 케톤 생성

---

### 칸 34: Claisen Condensation (클라이젠 축합)

**비유**: "에스터판 알돌 반응"

**반응**:
```
에스터 + 에스터 → β-keto ester + ROH
```

**메커니즘**:
1. **Step 1**: 에스터 α-탄소 탈양성자화 → 에놀레이트
2. **Step 2**: 다른 에스터 카르보닐에 친핵성 첨가
3. **Step 3**: RO⁻ 이탈 (1,2-제거)
4. **Final**: 온화한 산으로 중화

**결과**: β-keto ester (1,3-dicarbonyl)

---

### 칸 35: Dieckmann Condensation (딕만 축합)

**정의**: "분자 내 클라이젠 = 고리 형성"

```
다이에스터 → 고리형 β-keto ester
```

**조건**: 에스터 두 개가 같은 분자에 있어야 함
**결과**: 5원 또는 6원 고리 형성

---

## 🔷 중앙 열: CO₂, Malonic, Robinson Annulation

---

### 칸 36: Decarboxylation (탈카르복실화)

**비유**: CO₂가 "도망가는" 반응

**핵심 규칙**: 
> **"β-keto acids lose CO₂ upon heating"**
> (β-케토산은 가열하면 CO₂를 잃는다)

**왜 β-케토산만?**:
- 6원 고리 전이상태 형성 가능
- 에놀 형태로 CO₂ 탈락
- 이탈기가 공명 안정화된 에놀레이트

**예시**: Acetoacetic acid → 아세톤 + CO₂

---

### 칸 37: Malonic Ester Synthesis (말론산 에스터 합성)

**목적**: 치환된 카르복실산 합성

**5단계 과정**:
1. **Deprotonation**: 에놀레이트 형성 (두 에스터 사이의 H가 매우 산성)
2. **SN2**: 알킬 할라이드와 반응 → 새 C-C 결합
3. **Acidic hydrolysis**: 에스터 → 카르복실산
4. **Decarboxylation**: CO₂ 손실 → 에놀
5. **Tautomerization**: 에놀 → 카르복실산

**결과**: 원하는 R 그룹이 붙은 카르복실산!

---

### 칸 38: Robinson Annulation (로빈슨 고리화)

**비유**: "마이클 + 알돌 = 6원 고리"

**구성**:
1. **Michael Reaction**: 1,4-첨가 (공액 첨가)
2. **Intramolecular Aldol**: 분자 내 알돌 축합

**Part 1 - Michael Reaction**:
```
에놀레이트 + α,β-불포화 케톤 → 1,5-diketone
```

**Part 2 - Intramolecular Aldol**:
- 1,5-diketone이 분자 내 알돌
- **6원 고리** 선호 (5원 고리보다 안정)

**최종 생성물**: α,β-불포화 시클로헥세논

---

## 🔷 오른쪽 열: Haloform, HVZ

### 칸 39: Haloform Reaction (할로폼 반응)

**반응**: Methyl ketone + 과량 X₂/NaOH → 카르복실산 + CHX₃

**왜 특별한가?**:
- Methyl ketone만 해당 (CH₃-C=O)
- 3번 연속 할로겐화
- C-C 결합이 끊어짐!

**메커니즘**:
1. 에놀레이트 형성
2. 할로겐화 (3회 반복) → -CX₃ 형성
3. OH⁻ 공격
4. ⁻CX₃ 이탈 → CHX₃ (클로로폼, 브로모폼, 아이오도폼)

**응용**: 아이오도폼 테스트 (노란 침전 = methyl ketone 확인)

---

### 칸 40: α-Halogenation 비교

**Mono-, Di-, Tri-halogenation**:

| 생성물 | pKa | 특징 |
|-------|-----|------|
| 케톤 | ~19 | 출발물 |
| α-모노할로 | ~16 | 더 산성 |
| α-다이할로 | <16 | 더더 산성 |
| α-트라이할로 | - | 가장 산성, 에놀화 불가 |

**핵심**: 할로겐이 붙을수록 더 산성 → 다중 할로겐화 쉬움

---

### 칸 41: Hell-Volhard-Zelinsky (HVZ) Reaction

**목적**: 카르복실산의 α-탄소에 Br 도입

**시약**: 
- PBr₃ (1 equiv)
- Br₂ (1 equiv)  
- H₂O

**메커니즘 4단계**:
1. **Nucleophilic acyl substitution**: RCOOH + PBr₃ → RCOBr (산브로마이드)
2. **Keto-enol tautomerism**: 산브로마이드 → 에놀
3. **Bromination**: 에놀 + Br₂ → α-bromo acid bromide
4. **Hydrolysis**: 물로 가수분해 → α-bromo carboxylic acid

**응용**: 
- α-아미노산 합성의 전구체
- α-bromo 아마이드, 에스터로 전환 가능
